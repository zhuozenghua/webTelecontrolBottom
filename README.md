# web远程监控led数码管状态项目

## web 前端-app目录，由于项目简单，前端没有采用任何库和框架

### 前提你已经安装了npm/cnpm node

## 1.启动测试服务器
### wsServer.js为测试后台的文件

```
进入app目录
cd app

安装依赖
npm install

启动服务器（最多浏览器核数个进程）
npm run test
 or 
node wsServer
```

## 2.打开浏览器，直接访问index.html

## 3.打开控制台，我会打印发送的接口数据和接收的数据，同时可以观察后台控制窗口，也会打印详细信息

## web前端结束
